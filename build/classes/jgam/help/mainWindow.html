JGammon - Windows and Dialogs

The main window

The main window essentially consits of three parts: On top the menu bar, directly underneath the tool bar and then the actual backgammon board.

Das Spielfeld

Um Spielsteine zu bewegen, wird die Maus verwenden: Klicken Sie auf einen Stein und ziehen Sie ihn bei gedrückter Maustaste vom Ausgangszacken auf den Zielzacken. Sobald Sie einen Stein "aufgehoben" haben, bewegt er sich mit dem Mauszeiger und auf den Zacken, die als Ziel in Frage kommen, erscheint ein Vorschausymbol. (Dieses Verhalten kann auch abgeschaltet werden).

Alternativ kann auch auf einen Stein doppelgeklickt werden, dann wird der größtmögliche Einzelschritt mit diesem Stein gemacht.

Stein von der Schranke können auf dieselbe Weise wieder ins Spiel gebracht werden. Steine, die ausgespielt werden sollen, werden in das Ablagefeld neben dem Heimfeld gezogen.

Fenstergröße

Die Größe des Hauptfensters richtet sich nach dem gewählten Design. Da die meisten Designs auf Bitmap-Grafiken basieren, sehen sie in ihrer ursprünglichen Größe am besten aus. Dennoch lassen sie sich in gewissen Grenzen vergrößern und verkleinern. Dieses Verhalten kann aber auch im Menü deaktiviert werden.
