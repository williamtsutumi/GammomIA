JGammon - Einführung

Installation von JGammon

Damit JGammon auf Ihrem Rechner funktionieren kann, müssen folgende Bedingungen auf erfüllt sein:

  • Die Java Laufzeitumgebung JRE muss mindestens in der Version 1.4 installiert sein. Das JRE ist von der Firma SUN kostenlos für viele Betriebssyteme erhältlich ( java.sun.com).

Um eine Version JGammon zu installieren, gehen Sie folgendermaßen vor:

  1. Laden Sie eine neue Version von JGammon von der Homepage von JGammon herunter. Verweise und weitere Informationen erhalten sie unter jgam.sourceforge.net.
    Die aktuelle Version können Sie unter sourceforge.net/projects/jgam/files abrufen.
    Laden Sie die Datei JGammonRelease.zip, wobei Release für die aktuelle Version steht.
  2. Entpacken Sie die erhaltene Datei in ein neues Verzeichnis.
  3. JGammon ist installiert. Wenn Sie wollen, können Sie Verknüpfungen auf die uten erwähnten ausführbaren Dateien erstellen. Dies ist abhängig von Ihrem Betriebssystem oder Ihrer Oberfläche, schauen Sie in der Hilfe nach.

Starten von JGammon

Um JGammon zu starten, gibt es mehrere Möglichkeiten:

  • Öffnen Sie eine Eingabeaufforderung. Wechseln Sie in das Verzeichnis der JGammon-Installation. Geben Sie ein: 
       java -jar JGammon.jar
  • Windows: Doppelklicken Sie auf das Symbol von jgammon.
  • Unix/Linux: Starten Sie die jgammon.sh.

Das JGammon Hauptfenster erscheint.

JGammon Quellcode übersetzen

Wollen Sie die Dateien selbst übersetzen, müssen folgende frei verfügbaren Softwarepakete installiert sein:

  • ANT ( ant.apache.org) in einer Version 1.6 oder neuer
     
  • Java Development Kit in der Version 1.4 oder höher. Dieses kann wie das JRE bei SUN ( java.sun.com) heruntergeladen werden.

Laden Sie den Quellcode unter www.sourceforge.net/project/jgam/files herunter. Entpacken Sie die Datei JGammon.src.zip in ein neues Verzeichnis und starten Sie dort die Übersetzung mit
     ant jar

Dabei wird die Datei JGammon.jar erzeugt, die alle Daten für JGammon enthält und zum Starten verwendet werden kann.
