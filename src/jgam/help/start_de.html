JGammon - Einführung

Starten von JGammon

Damit JGammon auf Ihrem Rechner funktionieren kann, müssen folgende Bedingungen auf Ihrem Rechner erfüllt sein:

  • Die Java Laufzeitumgebung JRE muss mindestens in der Version 1.4 installiert sein. Das JRE ist von der Firma SUN kostenlos für viele Betriebssyteme erhältlich (java.sun.com).
     
  • Eine grafische Oberfläche muss laufen (XWindows, unter Windows selbstverständlich). Die Auflösung sollte mindestens 1024x768 betragen (muss aber nicht unbedingt)
Wollen Sie die Dateien selbst übersetzen, so wird ferner benötigt:
  • ANT (ant.apache.org) in einer Version 1.6 oder neuer
     
  • Java Development Kit in der Version 1.4 oder höher. Dieses kann wie das JRE bei SUN (java.sun.com) heruntergeladen werden.

Um eine Version JGammon zu installieren, gehen Sie folgendermaßen vor:

  1. Laden Sie eine neue Version von JGammon von der Homepage von JGammon herunter.Links und weitere Informationen erhalten sie unter jgam.sourceforge.net. Laden Sie entweder die übersetzten Daten (JGammonRelease.zip) oder den Quellcode (JGammonRelease.src.zip) herunter.
     
  2. Entpacken Sie die erhaltene Datei in ein neues Verzeichnis. Unter Linux geht das z.B mit unzip, unter Windows mit dem Explorer.
     
  3. Wenn Sie den Quelltext erhalten haben, öffnen Sie ein Kommandofenster (z.B. xterm unter Linux, Eingabeaufforderung cmd.exe unter Windows XP, ...) und geben Sie in dem Verzeichnis, das sie erstellt haben, ein[1]: 
       ant clean bindist
  4. In jedem Fall ist die erstellte oder heruntergeladene Datei JGammon.jar die entscheidende Datei.
     
  5. Um zu testen, ob JGammon gestartet werden kann, geben Sie in einem Kommandofenster im Verzeichnis dvon JGammon.jar ein: 
       java -jar JGammon.jar
  6. Das JGammon Hauptfenster erscheint.

[1] Falls Sie kein ANT installiert haben, so können Sie unter Linux mit javac `find . -name "*.java"` auch übersetzen. Es entsteht dann jedoch keine gültige JGammon.jar-Datei. Starten Sie daher dann JGammon mit java -cp src jgam.JGammon.
