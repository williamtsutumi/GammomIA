JGammon - Introduction

Playing the first game with JGammon

Once you have launched the JGammon program, the main window appears. Please see the explication of the main window and the toolbar for further info.

Start a local game

In order to start the game press the start button which is the left-most button in the toolbar.

In the upcoming dialog simply press OK. This dialog is discussed later in detail.

The initial board appears

The game has begun: The initial backgammon setup is set onto the board and the dices have been thrown to determine the starting player.

The player whose dice (colour!) has the higher number begins. You can always see whose turn it is in the toolbar.

Point the mouse cursor to the checker that you want to move, press the left mouse button and keep it pressed. Now you can drag around the checker. Drop it onto the point onto which you want to drop it.

Alternatively you can doubleclick on a checker than it will make the largest single step which is possible for it.

After you have made your moves according to the rules, it's the other party's turn.

Throwing the dice

After the first moves are done it is up to the next player to decide whether to double the game value or to throw the dice.

According to the rules, a player may only double if he/she was not the last to double or if nobody has doubled yet.

As it is very unusual to double already after the first move, we choose to throw the dice. Click the "THROW" button which is on the right of the toolbar.

The dice are thrown and the player can now make his/her moves.

Doubling the game value

Assume that after some more moves the position seems good for party so that he/she wants to double the game value. Instead of throwing the dice, press the doubling button ("2x") in the toolbar.

The other party has to accept the doubling offer or turn it down. A dialog pops up and asks whether the doubling is accepted (taken) or turned down (dropped).

Giving up

After some more moves the situation gets worse: I reckon I won't win this game at all. So I propose I give up to my opponent.

I may choose between normal, gammon and backgammon giveup. The opponent will be asked to accept/defeat the offer.

I can give up at any moment during the game, it doesn't have to be my turn to do so.

End of the game

Every games comes to an end eventually. If one of the players has moved the last checker to the off, the game is over.
