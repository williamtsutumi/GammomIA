The command line

If you startup JGammon from the command line you can provide arguments when starting up that will set up the game:

The available command line parameters are:

-connect <server>:<port>starts connecting to a server at a port
-server <port>starts as server listening to a port
-localstarts a local game
-aistarts a game againsz the artificial intelligence
-demoplays a demo game
-name <name>provide a name for a player (2 can be specified)
-board <file>sets the board file to be loaded
-design <number>the index of the desired design in the menu order (starting with 1)

A port-Argument needs to be a number between 1 and 65535, a server-Argument needs to be a valid computer address, either by IP or by name. Only for local games, two name arguments make sense. The first -name name1Argument sets the name for player 1 (accordingly the local player in network/AI games), a second -name name2 Argument sets the name of local player 2.

The values you provide on the command line are just filled into the corresponding fields of the new game dialog. All you hvae to do to launch the game is press ok.

These arguments are the same as can bentered into the new game dialog. Please see its documentation.
