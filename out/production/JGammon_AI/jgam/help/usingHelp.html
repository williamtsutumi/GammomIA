JGammon - Backgammon for Java

Welcome to the help system of JGammon

In these pages you will find information about the game backgammon, using JGammon and detail information on the program.

Layout of the help window

On the left of this help window you can see the two tabs "Content" and "Search". These are used to navigate through this help system.

With the content tree you can systematically browse the content of the help system. This content is hierarchically structured. The hierarchy can be expandend or reduced. Click on the icon in front of a folder to expand or reduce a topic.

With the index-panel you can look for specific topics by keywords. You can select the page which is associated with the cue entry.

In addition to the panel on the left, you can click links in this window, which are underlined words in blue. Links to external places are prefixed with and will be opened in a new browser window.

If your system is not automatically supported, you can specify the command to launch the browser manually by pressing the setup (world) symbol and entering the command to launch a browser on your system.

The bar between the navigation panel (left) and the content panel (right) can be moved left or right.
